# Stakeholder Insights and Performance Dashboard

## Project Overview
The **Stakeholder Insights and Performance Dashboard** is an interactive Tableau dashboard designed to provide a comprehensive view of key business metrics related to stakeholders, insurers, and policy performance.  
The dashboard enables decision-makers to analyze premium trends, profitability, asset distribution, and stakeholder contributions to support data-driven strategic decisions.

---

## Purpose
The primary purpose of this dashboard is to:
- Analyze insurer and stakeholder performance
- Track premium collection trends over time
- Evaluate profitability across regions and policy tenures
- Understand asset distribution and shareholding patterns
- Enable stakeholders to make informed business decisions through visual analytics

---

## Key Features

### 1. Charts and Visualizations
- Comparison of **total value created by different insurers**
- **Premium collection trends** over time
- **Shareholding pattern distribution** among stakeholders
- Relationship analysis between **cost ratio and profitability**
- **Profitability analysis by city and policy tenure**
- Distribution of **Assets Under Management (AUM)** across stakeholders

---

### 2. Key Performance Indicators (KPIs)
- **Total Premium Collected**
- **Average Age of Stakeholders**
- **Total Value Created** where Assets Under Management is *Equity*
- **Highest Value Created** by a single insurer
